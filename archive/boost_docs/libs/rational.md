# Rational Number Library

rational.hpp ヘッダは有理数を扱うための実装を提供する。
有理数ライブラリは、ちょうど一般的な複素数クラスのようにテンプレートベースで実装されている。

本ライブラリは一般的な用途を想定しており、数論への適用や、それ以外でも、数学的に厳密な演算が必要な場合には、もっと専門的な目的のライブラリを使うことを考えるべきである。
